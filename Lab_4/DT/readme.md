# Decision Tree (CART vs ID3) — Teens Mental Health Dataset

## Overview
This project trains, tunes and compares two decision tree algorithms — **CART** (Gini
impurity) and **ID3** (Information Gain / Entropy) — on the Teens Mental Health
dataset, with `depression_label` as the binary target. The full pipeline is automated
on GitHub: the dataset is fetched from a public raw URL, the trees are tuned with
`GridSearchCV`, and all required 2x1 side-by-side visualisations are produced without
any manual file upload.

## Dataset
- **Source file:** `dataset/Teen_Mental_Health_Dataset.csv`
- **Target:** `depression_label` (0 = Not Depressed, 1 = Depressed)
- **Features:** age, gender, daily_social_media_hours, platform_usage, sleep_hours,
  screen_time_before_sleep, academic_performance, physical_activity,
  social_interaction_level, stress_level, anxiety_level, addiction_level

## Implementation
- Two `DecisionTreeClassifier` estimators (Gini vs Entropy) from scikit-learn.
- Stratified 70 / 15 / 15 train / val / test split.
- `GridSearchCV` with 5-fold stratified CV and F1 scoring.
- Tuned hyperparameters: `max_depth`, `min_samples_split`, with `class_weight='balanced'`
  to mitigate the severe class imbalance.
- Side-by-side visualisations for both trees: decision boundary, confusion matrix,
  ROC curve, evaluation-metrics bar chart, and the optimised tree structure.

## Workflow
1. Load data from `dataset/Teen_Mental_Health_Dataset.csv` (with GitHub fallback).
2. Encode the three categorical columns with `LabelEncoder`.
3. Stratified train/val/test split.
4. Tune CART and ID3 with `GridSearchCV` on `max_depth` × `min_samples_split`.
5. Evaluate the optimised models on the test set.
6. Generate the five required visualisations.
7. Print the sample predictions and classification report.

when the target is highly separable by a small subset of features
(`anxiety_level` and `stress_level`).

## Notes
Decision trees are invariant to monotonic feature transformations, so feature scaling
is intentionally omitted. Both Gini and Entropy are mathematically similar impurity
measures — on a low-noise, low-dimensional dataset such as this one, the resulting
trees are nearly identical.
