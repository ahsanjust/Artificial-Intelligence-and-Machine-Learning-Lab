# Neural Network (ANN) — Shallow vs Deep — Teens Mental Health Dataset

## Overview
This project implements, tunes, and compares two distinct **Neural Network** architectures
built with **PyTorch** — a **Shallow NN** (exactly one hidden layer) and a **Deep NN**
(at least three hidden layers with dropout regularization) — on the Teens Mental Health
dataset for binary classification (`depression_label`). The full pipeline is automated
on GitHub: the dataset is fetched from a public raw URL, both models are tuned with
validation splits, and all required 2×1 side-by-side visualisations are produced without
any manual file upload.

## Dataset
- **Source file:** `dataset/Teen_Mental_Health_Dataset.csv`
- **Target:** `depression_label` (0 = Not Depressed, 1 = Depressed)
- **Features:** age, gender, daily_social_media_hours, platform_usage, sleep_hours,
  screen_time_before_sleep, academic_performance, physical_activity,
  social_interaction_level, stress_level, anxiety_level, addiction_level

## Implementation
- **Framework:** PyTorch (`torch.nn`, `torch.optim`)
- **Shallow NN:** Exactly one hidden layer. Hyperparameter-tuned on:
  - Hidden units: 16, 32, 64, 128
  - Activation functions: ReLU, Sigmoid, Tanh
  - Batch size: 16, 32, 64
- **Deep NN:** Three hidden layers with Dropout + Batch Normalization + L2 weight decay.
  Hyperparameter-tuned on:
  - Learning rate: 0.01, 0.001, 0.0001
  - Optimizer: Adam vs SGD (with momentum)
  - Epochs (up to 150)
- **Imbalance handling:** Weighted Binary Cross-Entropy (`pos_weight = n_neg / n_pos`)
- **Regularization (Deep):** Dropout (rate=0.3), Batch Normalization, L2 weight decay

## Workflow
1. Load data from `dataset/Teen_Mental_Health_Dataset.csv` (with GitHub fallback).
2. Perform exploratory data analysis (EDA).
3. Encode categorical columns with `LabelEncoder`.
4. Stratified 70/15/15 train/validation/test split.
5. Scale features with `StandardScaler`.
6. Build and tune the **Shallow NN** (1 hidden layer) with validation split.
7. Build and tune the **Deep NN** (3+ hidden layers) with dropout.
8. Evaluate both optimised models on the test set.
9. Generate all required visualisations (2×1 comparison matrices).

1. Open `220119_ANN.ipynb` in Jupyter or Google Colab.
2. Install dependencies: `pandas`, `numpy`, `matplotlib`, `seaborn`, `scikit-learn`, `torch`.
3. Click **Runtime → Run all**.
4. The dataset is fetched automatically — no manual file uploads required.

## Notes
- Both models use weighted binary cross-entropy to handle the ~50:1 class imbalance.
- The Deep NN includes Dropout, Batch Normalization, and L2 weight decay to combat
  overfitting on the small dataset (1 200 samples).
- Feature scaling (StandardScaler) is essential for neural network convergence.
